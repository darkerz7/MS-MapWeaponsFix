# Map Weapons Fix for ModSharp
Fixes weapon positions after the AG2 update

## Required packages:
1. [ModSharp](https://github.com/Kxnrl/modsharp-public)

## Installation:
1. Compile or copy MS-MapWeaponsFix to `sharp/modules/MS-MapWeaponsFix` folger

## CVARs:
Cvar | Parameter | Description
--- | --- | ---
`ms_mapweaponsfix` | <0/1> | Enable/Disable plugin
